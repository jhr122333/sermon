---

## 📖 About

매주 어린이 영어 예배 후 설교 요약, 기도문, 음성 파일을 한 페이지에서 확인할 수 있는 웹 아카이브입니다.

기존에는 PDF 유인물과 음성 파일이 따로 전달되어 아이들이 기도문을 들으며 따라 읽기 어려웠던 문제를 해결하기 위해 제작했습니다.

---

## ✨ Features

- 🎧 **오디오 플레이어** — 재생/일시정지, 진행바 클릭으로 이동
- 🐢 **재생 속도 조절** — 0.5x / 0.8x / 1x / 1.2x / 1.5x
- 🙏 **기도문 텍스트** — 음성 들으며 따라 읽기
- 📖 **스토리 & 핵심 교훈** — 설교 내용 요약
- 📄 **PDF 다운로드** — 유인물 원본 저장
- 📅 **날짜별 아카이브** — 지난 설교 모두 보관
- 📱 **앱 설치 없음** — 링크 클릭만으로 이용 가능

## 🔄 How to Update Weekly

매주 새 설교가 생기면:

1. 새 설교 HTML 파일 생성 (예: `sermon_mar15.html`)
2. 음성 파일 & PDF 업로드
3. `index.html`에 새 카드 추가 (맨 위에, Latest 뱃지 포함)
4. GitHub에 업로드 → 자동 배포
